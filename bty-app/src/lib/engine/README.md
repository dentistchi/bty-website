# bty Engine (A1)

목표: UI/콘텐츠가 바뀌어도 "기록/점수/승급"의 뼈대는 유지되는 코어 엔진을 먼저 고정한다.

## 핵심 개념
- **Core XP**: 개인의 장기 성장(절대치). 시즌이 바뀌어도 유지.
- **League XP**: 리그(기간 경쟁) 점수. 시즌 기간에 따라 집계.
- **Hidden Stats**: 화면에 직접 계산식을 노출하지 않고, 내부적으로만 누적되는 스탯(예: Gratitude/Integrity/Insight/Comms).
- **Stage**: 7개의 코드네임 * 100 단계 = 700까지는 이름 표시, **700+ 부터는 이름/숫자 숨김**.
- **Season**: 기간 고정(예: 2026-01-01~2026-03-31). Top 5% Elite는 시즌 단위로 산출.

A1에서는 DB/서버 통합 없이도 엔진 함수가 타입 안정적으로 동작하도록 설계한다.
