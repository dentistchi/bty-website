### 1-5. 언어 선택에 따른 시나리오·안내·대답 통일 (한국어/영어)

| 상태 | [x] |

```
bty-app에서 사용자가 한국어를 선택하면 한국어 시나리오, 영어를 선택하면 영어 시나리오가 나오도록 해줘. 안내 문구와 시스템 대답(피드백·메시지)도 선택한 언어와 같게 표시해줘.

- 확인: URL/경로의 locale(예: /ko/bty-arena, /en/bty-arena) 또는 사용자 설정·쿠키로 언어를 결정하고, 시나리오 데이터·i18n·API 응답에서 locale을 사용하는지 점검.
- 수정 범위: 시나리오 데이터(ko/en 필드 또는 별도 로케일 파일), getMessages(locale), Arena·리플렉션·free-response 등 API에서 locale 전달·응답 언어 분기. 기존 i18n 구조가 있으면 확장.
- 완료 기준: 한국어 선택 시 시나리오·안내·대답 전부 한국어, 영어 선택 시 전부 영어로 표시·응답됨.
```
