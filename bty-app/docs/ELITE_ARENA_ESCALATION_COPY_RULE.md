# Elite Arena — escalation text (content rule)

**Field:** `escalationBranches[<primaryChoiceId>].escalation_text` (canonical elite scenarios).

**Runtime:** This string appears as a **single compact line** above the forced tradeoff choices (primary choice → auto step 3 → forced tradeoff UI). It is **not** a separate screen.

## Rules

1. **Brief** — One or two short sentences. Aim for what fits comfortably in a small-type line block; avoid multi-paragraph copy.
2. **Non-redundant** — Do not restate the scenario setup, stem, or the learner’s primary choice. The learner already saw that on step 2.
3. **Decision-supportive** — Describe only the **new pressure, constraint, or shift** that makes the tradeoff meaningful *now*.
4. **No long narrative** — No backstory dumps, no dialogue blocks, no “meanwhile…” between primary choice and second choices.

**Check:** If you can delete a sentence without changing which option the learner should weigh, delete it.
