# 멘토 "깊이 파는 대화" 체감 방안

5% Elite가 전역 챗에서 멘토로 유도되는 흐름은 구현됨. 아래는 **멘토 페이지에서 "깊이"를 체감하게** 하기 위한 추가 기능 아이디어.

---

## 1. Elite 전용 표시 (이미 Elite만 멘토 유도됨)

- 멘토 페이지 상단에 **"Elite 전용"** 또는 **"상위 5% 대화실"** 뱃지/문구
- 진입 시 "주제를 골라 깊이 이야기해 보세요" 한 줄 안내

---

## 2. 대화 맥락 유지로 깊이 체감

- **멘토 API**: 현재 12턴(슬라이스 -12) 유지. Elite일 때 **20~24턴**으로 확대하면 "이전에 말한 걸 기억하고 이어서 묻는" 느낌 강화.
- 또는 **세션 단위 저장**: 같은 주제로 다시 들어왔을 때 "지난번에 ~ 이야기했었죠"로 이어주기 (구현 비용 높음).

---

## 3. 세션 마무리로 "한 문장" 추출

- 멘토 대화 **종료 시** (닫기/다른 주제로 이동 시):  
  "오늘 대화에서 가져갈 한 문장을 추천해 드릴까요?" → API로 마지막 N턴 요약 후 한 문장 생성.
- 사용자에게 **복사/저장** 버튼 제공 → "깊이 대화한 결과가 하나의 인사이트로 남는다" 체감.

---

## 4. 주제별 심화 질문 자동 제안

- 주제(임상/환자/팀/재무/자기사랑) 선택 후 **첫 메시지 전**에 "이 주제로 자주 묻는 질문" 2~3개 버튼 노출.
- 예: 임상 → "어려운 환자에게 설명할 때 어떻게 하면 좋을까요?", "실수했을 때 팀에 어떻게 말하면 좋을까요?"
- 클릭 시 그 질문이 첫 메시지로 들어가서 **바로 깊이 들어가는** 진입점 제공.

---

## 5. 대화 길이/깊이 인디케이터 (선택)

- 턴 수 또는 "대화 깊이" 단계(예: 1–3)를 작게 표시.  
  "이제 3단계 깊이예요" 같은 문구로 **이 대화가 짧은 질의응답이 아니라 심화 대화**라는 인지 강화.

---

## 구현 우선순위 제안

| 순서 | 항목 | 효과 | 비용 |
|------|------|------|------|
| 1 | Elite 뱃지/문구 (멘토 페이지) | 특혜 체감 | 낮음 |
| 2 | Elite일 때 멘토 API 컨텍스트 턴 수 확대 | 깊이 체감 | 낮음 |
| 3 | 주제별 "자주 묻는 질문" 버튼 | 진입 깊이 | 중간 |
| 4 | 세션 끝 "한 문장" 추천 | 결과물 체감 | 중간 |
| 5 | 세션 저장/이어하기 | 재방문 깊이 | 높음 |
