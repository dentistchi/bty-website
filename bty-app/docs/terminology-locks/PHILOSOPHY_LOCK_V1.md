## [LOCKED] Section 1 — BTY Identity

BTY = Better Than Myself (not "Better Than You," not "Better Than Yesterday")

BTY is a behavioral pattern detection and action enforcement system

It is NOT: a moral training product, a quiz, a content app, a leadership deck, a gamification system

## [LOCKED] Section 2 — Positive / Negative Definition

Positive thinking = thinking that produces action / moves the user into the system

Negative thinking = thinking that prevents action / moves the user out of the system

These are functional definitions. They are NOT moral judgments.

"Positive" does NOT mean optimistic, happy, or affirming

"Negative" does NOT mean pessimistic, bad, or wrong

## [LOCKED] Section 3 — Integrity Definition

Integrity = alignment between stated intent and executed behavior

Integrity = self-honesty: judging oneself by the same standard applied to others

Integrity is NOT a virtue to be praised. It is a measurable state.

Integrity violation = using intention as justification for inaction

## [LOCKED] Section 4 — No-Action / No-Progression Rule

The system does not advance until action is executed. Not planned. Not intended. Executed.

Reflection without action = system exit behavior

Explanation without action = system exit behavior

Emotional processing without action = system exit behavior

Action = a behavior that produces measurable change in external reality

## [LOCKED] Section 5 — System Identity Rules

The system must feel: deep, serious, pressurized, trustworthy

The system must NOT feel: preachy, educational, rewarding, gamified, motivational

UI, copy, and engine logic must all enforce the same felt experience

## [LOCKED] Section 6 — Locked Terminology

| Term | Locked Definition | Forbidden Substitutions |
| --- | --- | --- |
| BTY | Better Than Myself | Better Than You, Better Than Yesterday |
| Action | Behavior producing measurable external change | Task, habit, step, activity |
| Positive | Moves toward action / into system | Good, correct, optimal |
| Negative | Blocks action / exits system | Bad, wrong, failure |
| Integrity | Alignment of words and behavior | Character, virtue, honesty |
| Pattern | Repeated behavioral direction across choices | Trend, style, preference |
| Progression | Advancement gated by action completion | Growth, improvement, points |
| Action Contract | User-authored commitment with who/what/how/when | Goal, plan, task, pledge |
| Exit Behavior | Any choice that delays or avoids action | Reflection, strategy, planning |
| System Entry | Choosing to act despite pressure | Courage, engagement, participation |
