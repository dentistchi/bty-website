# MVP 배포 준비 — 배포 전 1회 체크리스트

**목적:** MVP를 실제로 배포하기 **직전**에 1회 수행하는 점검.  
**정책:** 일상 작업은 웹 개발(UI·API·도메인) 집중. 이 체크리스트는 **배포를 결정한 시점**에 한 번 실행한다.

---

## 사용 방법

1. 배포할 준비가 되었을 때만 이 문서를 연다.
2. 아래 항목을 순서대로 확인하고, 완료 시 `[ ]` → `[x]`로 바꾼다.
3. 모두 통과하면 MVP 배포 진행. 실패 항목은 수정 후 다시 점검.
