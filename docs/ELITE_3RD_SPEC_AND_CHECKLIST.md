## 1. 스펙 요약

| 항목 | API·도메인 | UI |
|------|------------|-----|

## 2. 검증 체크리스트 (6항)

| # | 항목 | 확인 |
|---|------|------|
| 1 | 배지 API 계약 | GET `/api/me/elite`에 `badges` 배열, Elite일 때만 비어 있지 않음. |
| 2 | 배지 UI 노출 | `/bty/elite`·대시보드 Elite 카드에서 `badges` 표시. 비Elite 시 "상위 5% 달성 시"만. |
| 3 | 멘토 신청 API | POST/GET `/api/me/mentor-request`, GET/PATCH `/api/arena/mentor-requests` (admin). |

| 5 | 규칙 준수 | XP/랭킹/시즌 계산 없음. domain·API·bty-ui-render-only·bty-arena-global 준수. |
| 6 | 경로 회귀 없음 | `/bty/elite`, `/bty/dashboard`, `/bty/mentor`, `/admin/mentor-requests` 정상. |

**PASS**: 전 항목 충족. **FAIL**: 불충족 번호 + 현상 기록.
