# BTY (Better Than Yesterday) MASTER PLAN

## 1. SYSTEM OVERVIEW
BTY는 조직 문화를 확장하고 정직성(Integrity)을 표준화하도록 설계된 결정론적 리더십 엔진(Deterministic Leadership Engine)이자 행동 정렬 시스템입니다. 본 시스템은 단순한 동기부여 플랫폼이 아닌, 의도를 검증 가능한 행동으로 전환하여 문화적 변질을 방지하는 객관적 자동화 시스템입니다.

## 2. CORE ARCHITECTURE
시스템은 상호작용하는 세 가지 엄격한 경계 환경에서 작동합니다.

- Arena (Execution): 실시간 시뮬레이션 환경. 압박감 속에서의 의사결정을 통해 XP를 획득하고 리더보드에서 경쟁하며, 행동 패턴의 1차 진단 레이어 역할을 합니다.
- Foundry (Development): 교육 및 철학 허브. 12개의 학습 프로그램, AI 멘토(Dr. Chi), Dojo 기술 평가를 통해 리더의 행동 품질을 향상시킵니다.
- Center (Calibration): 비공개 진단 및 회복 환경. 'Dear Me' 편지, 탄력성 추적, 자존감 평가를 통해 정서적 안정과 Integrity Reset 프로토콜을 수행합니다.

[Interaction Loop]: Arena → Foundry → Arena 순환 구조를 강제하며, 기준 미달 시 Center로 퇴출(Ejection)되어 복귀 절차를 거쳐야 합니다.

## 3. CORE METRICS SYSTEM
- AIR (Action, Integrity, Responsibility): 행동 중심 지표. 기준 미달 시 'Integrity Slip' 발생.
- TII (Team Integrity Index): 팀 건강도 (60% AIR + 25% MWD + 15% TSP). 개인 보호를 위해 팀 단위만 공개.
- LRI (Leadership Readiness Index): 14일간의 데이터를 기반으로 한 승진 준비도.
- Certified Leader: 14일간 AIR 80% 이상 유지 시 부여되는 휘발성 자격 (90일마다 갱신).

XP System:
- Core XP (영구적 성장/ID 결정)
- Weekly XP (시즌제 리더보드 전용)
- Lab XP (연습용, Core XP만 지급)

## 4. USER FLOW
1. Entry & Execution: Arena에서 갈등 시나리오 해결 및 즉각적인 XP 획득.
2. Pattern Detection: 행동 엔진이 'Hero Trap' 등의 결함 감지 및 기록.
3. Consequence & Mirroring: 반복된 왜곡 시 '지연된 결과' 또는 '역할 미러링' 시나리오 강제 실행.
4. Growth (Foundry): 일관성 증명 시 교육 허브 해제 및 AI 멘토링 접근.
5. Correction (Center): AIR 급락 시 48시간 락아웃 및 Center 진단 통과 후 Arena 복귀.
